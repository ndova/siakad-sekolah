### 4.2. Narasi Alur Utama

#### A. Login
1. Siswa membuka `siswa.sekolah.sch.id` (atau mobile PWA)
2. Input NIS + password → validasi server → redirect ke Dashboard

#### B. Dashboard (Halaman Mendarat)
3. Dashboard menampilkan **6 card ringkasan** real-time
4. Siswa melihat: jadwal hari ini, deadline ujian, tagihan SPP, progress P5, presensi
5. **Panel kanan**: notifikasi prioritas tinggi + quick actions

#### C. Mengikuti Ujian Online
6. Dari Dashboard → klik "Ujian Terdekat" atau sidebar "Ujian Online"
7. Halaman list ujian: lihat status (tersedia/terjadwal/selesai)
8. Klik **"Mulai Ujian"** → dialog konfirmasi aturan ujian
9. Setuju → **full-screen mode** aktif, timer mulai
10. Mengerjakan: navigasi soal via panel kanan, jawab PG/esai, tandai ragu
11. Auto-save setiap 30 detik. Koneksi putus → notifikasi, sync saat online kembali
12. **Submit** → konfirmasi sisa soal → lihat hasil langsung (PG auto-graded, esai menunggu koreksi guru)
13. Beberapa saat kemudian → hasil final tersedia di "Riwayat Ujian"

#### D. Melihat Nilai & Rapor
14. Sidebar **"Nilai & Rapor"** → 3 tab: Intrakurikuler, P5, Rapor
15. Tab intrakurikuler: tabel nilai per mapel + status Tuntas/Remedial
16. Klik mapel → expand detail per **Tujuan Pembelajaran (TP)**
17. Tab Rapor → pratinjau PDF atau download rapor format Kemendikbudristek
18. Rapor meliputi: nilai intrakurikuler, P5, ekstrakurikuler, presensi, catatan wali kelas

#### E. Memantau & Membayar SPP
19. Sidebar **"Pembayaran"** → daftar tagihan aktif + status
20. Jika ada tunggakan → badge peringatan di dashboard dan sidebar
21. Klik **"Bayar Sekarang"** → pilih metode: VA BNI, QRIS, transfer, e-wallet
22. Sistem generate nomor VA unik per tagihan → tampilkan instruksi
23. Siswa/orang tua bayar via m-banking/ATM/e-wallet
24. Payment gateway callback → status otomatis **LUNAS** (1-5 menit)
25. Riwayat pembayaran + kwitansi bisa dilihat dan di-download

## 5. Elemen UI Penting & Design System

### 5.1. Komponen Global

| Komponen | Fungsi | Posisi |
|---|---|---|
| **Top Bar (sticky)** | Info user, notif bell, hamburger (mobile) | Paling atas, selalu terlihat |
| **Sidebar Navigasi** | Menu utama + avatar + nama/kelas | Kiri, collapse di tablet |
| **Breadcrumb** | Navigasi hierarki | Atas content area |
| **Panel Kanan** | Notifikasi, ujian terdekat, quick links | Kanan, hanya desktop |
| **Footer** | Copyright, versi, link bantuan | Bawah content |

### 5.2. Komponen Spesifik

| Komponen | Deskripsi |
|---|---|
| **Countdown Timer** | Real-time hitung mundur untuk ujian (merah jika <5 menit) |
| **Progress Bar** | Untuk P5, nilai mapel, kelengkapan data |
| **Status Badge** | 🟢 Lunas, 🔴 Jatuh Tempo, 🟡 Izin, 🟢 Tuntas, 🟡 Remedial |
| **Calendar Grid** | Untuk jadwal & presensi, highlight hari ini |
| **Tabel Nilai** | Sortable, expandable row untuk detail TP |
| **Nomor Soal Grid** | 7 kolom, warna hijau/putih/kuning, clickable |
| **Pagination / Load More** | Untuk riwayat panjang (ujian, pembayaran, presensi) |
| **Modal Konfirmasi** | Untuk submit ujian, mulai ujian, logout |
| **Toast Notifikasi** | Sukses/gagal di pojok kanan atas, auto-dismiss |
| **Skeleton Loader** | Placeholder shimmer saat data loading |

### 5.3. Notifikasi

| Tipe | Channel | Contoh |
|---|---|---|
| **In-App (bell icon)** | Panel kanan + drawer | "Ujian Matematika dimulai besok" |
| **Browser Push** | Service Worker | "Tagihan SPP jatuh tempo" |
| **Email** (opsional) | Ke email orang tua | Rapor sudah bisa diunduh |
| **WA Gateway** (opsional) | Ke nomor orang tua | Pembayaran terkonfirmasi |

### 5.4. Responsivitas & Mobile

- **PWA (Progressive Web App)** — bisa di-install di homescreen HP
- **Touch-friendly**: tombol min 44x44px, swipe gesture untuk navigasi
- **Mobile ujian**: layout single-column, panel soal penuh, navigasi soal di drawer bawah
- **Offline support**: cache jadwal & nilai terakhir, sync saat online

### 5.5. Aksesibilitas

- Semantic HTML (`<nav>`, `<main>`, `<article>`, `<table>`)
- ARIA labels pada interactive elements
- Kontras warna WCAG AA (min 4.5:1)
- Keyboard navigable (Tab, Enter, Arrow keys untuk navigasi soal)
- Screen-reader friendly untuk siswa dengan kebutuhan khusus
